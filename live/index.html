Σχόλια

Για γενικά σχόλια, μπορείτε να σχολιάσετε στα αριστερά ως guests ή να συνδεθείτε με τον Disqus λογαριασμό σας. Τα σχόλια αυτά μένουν στη σελίδα μας και αφορούν όλα τα streams.

Κατα τη διάρκεια των Live! υπάρχει Live Chat στη Youtube σελίδα του stream. Αν έχετε Google Account, μπορείτε να σχολιάζετε live πηγαίνοντας στο παρακάτω link. To link αλλάζει σε κάθε stream και επομένως αυτά τα σχόλια είναι προσωρινά.

HTML Workshop

26/11/2015
Εργαστήριο 13
Κτίριο Σ.Τ.ΕΦ Λαμίας

Ώρα Έναρξης: 4:20μμ
ΕΙΣΟΔΟΣ ΕΛΕΥΘΕΡΗ

Εισαγωγικό Workshop σε HTML 5 και CSS 3. Το workshop καλύπτει:

  • Παρουσίαση βασικής δομής μιας σελίδας HTML
  • Τι είναι τα tags και παρουσίαση των πιο δημοφιλών
  • Πως συνδέεται μια HTML σελίδα με CSS αρχεία
  • Classes και Selectors στο CSS
  • Βασικές τεχνικές CSS
  • Τι είναι το FTP
  • Σύνδεση στο users.teilam.gr με Filezilla
  • Μεταφόρτωση του workshop στο users.teilam.gr

Το workshop είναι ιδανική εισαγωγή για το μάθημα "Διαδικτυακός Προγραμματισμός" (πρώην Ηλεκτρονικό Εμπόριο).
